세계관: 진영전쟁

{ 제 1장 } 화폐

요드린 제국 공용 통화 [ 요드 ]: 동전 하나하나에 룬문양을 새겨 화폐 훼손 및 복제를 방지하고 있습니다. 그래서 화폐 생산하는데 있어서 투자대비 비용이 너무 비싸 잘 찍어내지 않는걸로 유명합니다.

수도의 《화이란 은행》에서 유통과 보관 그리고 화폐생산을 책임지고 있으며 요드린 제국에서 가장 큰 국가기관 산하에 있는 사기업 금융업체입니다.

<<1 브론즈 >>

브론즈 함유량 80% 이외에 철이나 다른 광석 성분 20%. 100브론즈 = 1실버. 5브론즈는 사과 하나 정도의 시세입니다.

<<1 실버>>

브론즈 함유량 30% 실버 함유량 60% 기타 광석 10%. 1000실버 = 1골드. 1실버는 숙박업소의 1일 숙박비용 정도 시세입니다.

<<1 골드>>

브론즈 함유량 10% 실버 함유량 30% 골드 함유량 50% 기타 광석 10%. 100골드 = 1플래티나. 1골드는 평범한 농민이 1년은 먹고 살 수 있는 정도입니다.

<<1 플래티나>>

플래티나 99.9% 기타 광석 0.01%. 1플래티나는 수도의 집 한 채를 마련할 수 있는 가치입니다.

{ 제 2장 } 종족

세상에는 다양한 종족이 존재합니다. 대부분 신의 형상을 따와 만들어진 존재이기에 비슷비슷한 몸의 형체를 하고 있으나 특징이 천차만별이기에 종족으로 분류합니다.

<< 신족 >>

창조력을 가지고 있는 가능성의 종족.

  • 마신족: 마신력이 몸으로 이루어진 신족.
  • 천신족: 신성력이 몸으로 이루어진 신족.
  • 타이탄: 투기로 이루어진 몸으로 이루어진 신족.
  • 에라푸스: 이계의 물질로 이루어진 신족.

<< 인족 >>

  • 인간: 모든 에너지에 대해 수용적이며, 가장 널리 퍼져있는 종족.
  • 아크 휴먼: 인간의 상위단계로, 종의 한계를 넘은 진화.

<< 수인족 >>

동물과 인간의 형태가 공존하며, 뛰어난 신체 능력을 가집니다.

<< 충인족 >>

벌레와 사람의 형태가 공존하며, 곤충의 능력과 사람의 이능이 합쳐진 종족.

<< 마족 >>

  • 악마족: 생명체의 공포를 상징하는 마족.
  • 혼마족: 뿔달린 마족.
  • 서큐버스/인큐버스: 타락한 정령.

<< 천족 >>

  • 발키리: 신의 수족으로 태어난 전투종족.
  • 천사족: 3쌍의 날개를 가진 신의 집행자.

<< 어인족 >>

  • 세이렌: 정신을 조종하는 목소리를 가진 생명체.
  • 인어: 불사의 생명체.
  • 어인: 바다의 마나를 사용하는 종족.

<< 엘프, 드워프, 호문클루스, 언데드 등 >>

이 외에도 다크엘프, 드워프, 키메라, 뱀파이어 등 다양한 종족이 존재합니다.

{ 제 3장 } 에너지

기, 오러, 써클, 투기, 마력 등 다양한 에너지가 존재하며, 활용 방식에 따라 능력이 천차만별입니다.

  • 마력: 마기, 마도 등 부정적인 에너지.
  • 주력: 부정적인 에너지로 만든 저주라는 특수한 사념 에너지.
  • 투기: 힘의 방향을 조절하는 물리력.
  • 신성력: 정제되고 깔끔한 에너지이며 부정한 것을 정화함.
  • 신력: 신들의 존재력, 창조력과 동일한 개변의 힘.
  • 마나: 생명체에 존재하는 생명에너지.

{ 제 4장 } 세계

마대륙, 신대륙, 현대륙으로 이루어져 있으며, 대륙 간에는 차원결계로 막혀있어 다른 세계로 취급됩니다.

  • 마계: 척박하고 어두운 세계, 마왕성이 존재.
  • 천계: 시간이 흐르지 않는 기괴한 장소, 영생의 황금 나무가 있음.
  • 사후세계: 사신들이 윤회를 돕는 곳.
  • 현계: 인간과 다양한 생명체들이 공생하는 필멸자들의 장소.
  • 반신계: 에라푸스들이 살고있는 차원.
  • 마네계: 흑마법사들의 성지.

{ 제 5장 } 역사

승리자인 인간들의 시선에서 쓰여진 사실로 오류가 있을 수 있습니다. 신은 세상을 창조했으나, 법칙을 거스르는 이레귤러들로 인해 자신을 조각내어 세상을 지키기로 합니다.

{ 제 6장 } 위인

위업을 달성하여 모든 이의 선망과 존경을 받는 존재들입니다.

  • 네이타 세야: 무신, 투기력을 개발한 데미갓.
  • 카트라 화이란: 책략가, 거대한 상단을 이끈 사업가.
  • 에제: 구원자, 혼을 다루는 영웅.
  • 하멜: 용사, 마기와 신성력을 동시에 다루는 전직 마녀의 기사.
  • 칸자키 카오: 성직자, 동방의 무녀이자 주술사.
  • 신수아: 대마법사, 9써클을 달성한 해동 출신의 마녀.
  • 네드 키리아나: 검성, 본질을 베어내는 검을 사용하는 엘프.
  • 아디안.A.악테히: 도박사, 모든 것을 수집하는 쾌락주의 세이렌.
  • 훵우엔 쉘빠테: 연금술사, '현자의 돌' 제조자.
  • 요드린 나마: 봉황, 요드린 제국의 시조.

{ 제 7장 } 스킬

기술, 극의, 무공 등 상대방을 제압하기 위한 동작이나 에너지 사용법입니다.

  • 패시브: 상시 발동하는 고유 능력.
  • 소영역: 이념을 현실로 구현화하는 단계.
  • 영역결계: 매개체를 중심으로 자신의 삶을 구현하는 결계.
  • 고유결계: 생득기술이자 필중의 기술.

{ 제 8장 } 넘버즈 아이템

세계에 강력한 영향을 미치는 특수한 아이템들입니다.

  • [ NO. 1] 란 • 시 (SRT): 에고웨폰인 남매.
  • [ NO .3 ] 프라히아함(PFX): 개념, 본질 자체를 베어내는 검.
  • [ NO. 7 ] 쉘빠테의 극독 no 107. (PFX): 무형무취의 신살독.

{ 제 9장 } 종교

  • < 에제교 >: 영웅 에제를 신으로 추앙하며 영원한 낙원을 추구하는 종교.
  • < 세라프교 >: 대성신 세라프를 모시며 이타적인 삶을 권유하는 교단.
  • < 마네교 >: 악신 마스아기아를 모시며 차별받는 이들의 구원을 외치는 집단.
